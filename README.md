# ProPhex

[![Build Status](https://travis-ci.org/prophyle/prophex.svg?branch=master)](https://travis-ci.org/prophyle/prophex)

## Prerequisities

* GCC 4.8+ or equivalent
* ZLib


## Getting started

```
git clone --recursive https://github.com/prophyle/prophex
cd prophex && make -j
```
